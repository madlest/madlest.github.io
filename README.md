# Madlest GitHub Pages

This repository hosts a small personal website for Madlest.

## Pages

| File | Purpose | Notes |
|---|---|---|
| `index.html` | Main profile page | Personal introduction and site navigation |
| `ref.html` | OC reference page | Original character details and image gallery |
| `comms.html` | Commissions page | Vocal tuning/timing details, pricing, samples, and queue link |

## Notes

- All pages are static HTML/CSS files.
- The site is designed for GitHub Pages and can be viewed directly from the repository root.
- A back link to `index.html` is included on both `ref.html` and `comms.html` for easier navigation.
